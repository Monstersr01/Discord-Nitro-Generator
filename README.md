## HOW TO GET SESSION ID

1. Go to https://www.opera.com/gx/discord-nitro on OperaGX browser
2. Create an account
3. Go back after creating the account
4. Now, Right click it will open Devtools
5. Click on Network
6. Click on Claim Now
6. It will open a new url, go back to the other one
7. Find for the requests named "token"
8. Go down until you find the Cookie headers and take the session_id
9. Here you go!

## Overview 
 
This script automates the generation of Discord Nitro URLs by making requests to the Discord OperaGX Nitro API. It's a simple and efficient way to create multiple Nitro URLs quickly.

## Usage 
  
1. **Clone the repository:**
   ```bash          
   git clone https://github.com/TheCuteOwl/Discord-Nitro-Generator.git 
   cd Discord-Nitro-Generator-main 
   ``` 

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the script:**

   ```bash 
   python main.py
   ```

   Follow the on-screen instructions to input the number of Nitro URLs you want to generate.
